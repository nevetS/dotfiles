# bin

Helper scripts for managing dotfiles.

| Script              | Platform      | Purpose                                      |
|---------------------|---------------|----------------------------------------------|
| `install-deps.sh`   | Linux, macOS  | Install python3 and rclone                   |
| `install-deps.ps1`  | Windows       | Install python3 and rclone via winget        |

## Adding scripts

Place any cross-machine utility scripts here. If a script is platform-specific,
name it clearly (`foo.sh` / `foo.ps1`) and note it in this table.
